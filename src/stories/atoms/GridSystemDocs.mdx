The basic GridSystem is made up of:
* `12` columns in `xs`
* `12` columns from `sm` to `md`
* `12` columns from `lg` up

```json
{
  // Number of columns in Grid based on Breakpoints
  columns: {
    xs: 12,
    sm: 12,
    md: 12,
    lg: 12,
    xl: 12,
    xxl: 12,
  },
  // `padding-left` and `padding-right` of `<Col />` based on Breakpoints
  gutterWidth: {
    xs: 16,
    sm: 16,
    md: 24,
    lg: 24,
    xl: 24,
    xxl: 24,
  },
  // `padding-left` and `padding-right` of `<Container />` based on Breakpoints
  paddingWidth: {
    xs: 16,
    sm: 16,
    md: 24,
    lg: 24,
    xl: 24,
    xxl: 24,
  },
  // `<Container />` max width based on Breakpoints
  container: {
    xs: "100%",
    sm: 540,
    md: 720,
    lg: 960,
    xl: 1140,
    xxl: 1280,
  }
}
```

| ---- | ---- | ------- | ----------- |

| `debug` | boolean | `undefined` | Background debug |

## `<Col />`

### Props
| Prop | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| `xs` | number or string | `12` | Set the number of columns on `xs` screen |
| `sm` | number or string | `12` | Set the number of columns on `sm` screen |
| `md` | number or string | `12` | Set the number of columns on `md` screen |
| `lg` | number or string | `12` | Set the number of columns on `lg` screen |
| `xl` | number or string | `12` | Set the number of columns on `xl` screen |
| `xxl` | number or string | `12` | Set the number of columns on `xxl` screen |
| `$direction` | string or object | `column` | Set `flex-direction` css property for the content below |
| `justify` | string or object | `undefined` | Set `justify-content` css property for the content below |
| `align` | string or object | `undefined` | Set `align-items` css property for the content below |
| `$order` | number or object | `undefined` | Set `order` css property |
| `$offset` | number or object | `undefined` | Set the number of the left offset columns on the left |
| `offsetRight` | number or object | `undefined` | Set the number of the offset columns on the right |
| `reverse` | boolean or array | `undefined` | Reverse the direction for the content below |
| `noGutter` | boolean | `undefined` | Remove paddind on left and right side: `gutterWidth` |
| `debug` | boolean | `undefined` | Background debug |
| `children` | node |  | Content |

### Code
<Source
  dark
  language="jsx"
  type="code"
  code={`
<Col
  xs={2}
  sm={2}
  md={4}
  lg={4}
  xl={6}
  xxl={6}
  $direction="row" or $direction={{ xs: "column", lg: "row" }}
  justify="center" or justify={{ xs: "flex-start", lg: "center" }}
  align="center" or align={{ xs: "flex-start", lg: "center" }}
  $order={2} or $order={{ xs: 2, lg: 3 }}
  $offset={2} or $offset={{ xs: 4, lg: 6 }}
  offsetRight={2} or offsetRight={{ xs: 4, lg: 6 }}
  reverse or reverse={["xs", "sm"]}
  noGutter
  debug
/>
  `}
/>
